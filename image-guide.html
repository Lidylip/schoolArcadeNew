🖼️ 游戏图片快速获取指南

🤖 方法1: 自动截图 (最推荐)

在添加游戏工具中直接使用自动截图功能，最方便快捷！

📋 使用步骤:

  1. 在"添加游戏"页面填写游戏名称和iframe链接
  2. 选择"🤖 自动截图"选项
  3. 点击"📸 生成截图"按钮
  4. 系统自动生成游戏截图

优点: 完全自动化，无需手动操作，图片尺寸标准(400x300)，质量一致

📸 方法2: 手动截图上传

🖥️ 电脑端截图:

  • Windows: Win + Shift + S
  • Mac: Cmd + Shift + 4
  • Chrome: F12 → 设备模拟器截图

🔍 方法3: 从游戏源站获取

🎯 常用方法:

  1. 右键检查: 在游戏页面右键"检查元素"，找到图片标签复制链接
  2. 开发者工具: F12 → Network → 刷新页面 → 找到图片文件
  3. 查看源码: Ctrl+U 搜索 ".jpg" 或 ".png"
  4. 谷歌图片: 搜索"游戏名 + screenshot"获取高质量截图

💡 实用技巧

✅ 推荐做法:

  • 图片尺寸: 400x300 或 16:9 比例
  • 文件格式: JPG/PNG
  • 文件大小: 500KB以内
  • 图片内容: 游戏实际画面截图
  • 避免包含UI界面，专注游戏内容

❌ 避免事项:

  • 避免使用过大的图片文件
  • 不要使用有版权争议的图片
  • 避免模糊或低质量图片
  • 不要使用不相关的图片
  • 避免包含个人信息的截图
